## 项目职责
* 设计机械臂结构，包括**固定底座，旋转底座，连杆机构，夹爪机构**
* 3D打印零部件，并进行机械组装
* 绘制主控板，包括**供电，控制器，串口，排针等等**
* 编写机械臂控制代码，包括**舵机驱动代码，蓝牙驱动代码**
* 机械臂运动学算法，**正逆运算**
